CPF:
- Deve ser informado como string e somente números (Ex: 11111111111) que vão ser salvos como char[] de forma correta (Ex: 111.111.111-11)
- Os números informados devem ser de um CPF válido

Endereco:
- Deve ser informado o CEP como string e somente números (Ex: 11111111) que vão ser salvos como char[] de forma correta (Ex: 11111-111)
- Os números informados devem ser de um CEP válido

Telefone:
- Deve ser informado como string e somente números (Ex: 99999999999) que vão ser salvos como char[] de forma correta (Ex: (99)99999-9999)
- Pode ser informado qualquer número
